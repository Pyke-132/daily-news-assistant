# AI Coding Agent Rules

1. Do not use Docker for this project.
2. Do not create a frontend.
3. Do not introduce heavy frameworks.
4. Do not delete `news.db`, `outputs/`, or `logs/` unless the user explicitly allows it.
5. After modifying code, run `uv run python main.py` or at least run a syntax check.
6. Every new configuration option must be documented in `README.md`.
7. Never write real API keys into code or `README.md`.
